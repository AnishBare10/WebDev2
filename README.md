SocialStream Analytics is a high-performance, executive-grade command center engineered for modern digital strategists and brand managers. In an era where digital presence is fragmented across multiple ecosystems, this platform provides a centralized telemetry hub to monitor, analyze, and orchestrate brand narratives with surgical precision. It transforms raw social data into actionable intelligence, allowing users to move beyond simple posting toward a comprehensive, data-driven cross-platform strategy.

The core architecture of the platform revolves around real-time visibility and strategic deployment. Through the Executive Dashboard, users gain an instantaneous overview of global reach and growth velocity across Instagram, Twitter, LinkedIn, and Facebook. The Advanced Analytics module dives deeper, utilizing sophisticated visualization engines to map engagement patterns, demographic distributions, and optimal performance windows. This level of granular detail ensures that every piece of content is backed by historical performance data and geographic heatmaps.

Efficiency is at the heart of the SocialStream experience, highlighted by the integrated Content Orchestrator. This module streamlines the creative workflow by offering a high-fidelity simulation environment where users can design and preview content for specific platform nodes before scheduling them for broadcast. The Transmission Queue provides a robust staging area for multi-channel campaigns, ensuring that brand messaging remains consistent and timely across the entire digital frontier.

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
